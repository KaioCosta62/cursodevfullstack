Quando usamos o display flex em um elemento, todos os elementos filhos vão ficar lado a lado

Flex-Direction: - Define a direção dos elementos filhos, por padrão o valor é row(em linha)

Flex-Direction: row - É o valor padrão, deixa os elementos em linha um ao lado do outro

Flex-Direction: row-reverse - Leva os elementos para o lado direito e inverte a ordem de exibição

Flex-Direction: column - Deixa os elementos filhos um embaixo do outro, em coluna

Flex-Direction: column-reverse - Deixa os elementos filhos um embaixo do outro, em coluna, mas inverte a ordem de exibição



Flex-Wrap: Por padrão o valor do flex-wrap é no-wrap, então os elementos não irá quebrar pra baixo, vão permanecer juntos na mesma linha, ficando cada vez mais espremidos

Flex-wrap: wrap - Quando o valor de flex-wrap é wrap, sempre que um conteúdo não couber na tela, irá cair para a linha de baixo, isso já é um princípio de design responsivo

Flex-wrap: wrap-reverse - Mesmo comportamento do wrap, pórem os conteúdos vai quebrando em ordem inversa



Flex-Flow: É um shorthand para a propriedade flex-direction e flex-wrap
